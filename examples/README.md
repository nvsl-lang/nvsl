# NVSL Examples

This directory contains examples for embedding and extending NVSL.

## [Host Integration](./host-integration)

Shows how to:
1.  **Register a custom library** (like `vn.*`) using `ScriptHost.registerSimple`.
2.  **Expose engine-specific logic** (like a renderer) to NVSL scripts.
3.  **Execute the script** from Haxe and see the side effects in your engine.

### Running this example:

From the project root, run:

```bash
haxe examples/host-integration/run-example.hxml
```
